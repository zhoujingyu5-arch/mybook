# TWAP 交易考核机制分析与最优策略

## 一、考核机制理解

### 1.1 核心指标
```
日盈亏百分比 = Σ(单只股票盈亏金额) / 当日总成交金额 × 100%

其中：
单只股票盈亏金额 = (市场均价 - 成交均价) × 成交量（买入指令）
                  或 (成交均价 - 市场均价) × 成交量（卖出指令）
```

### 1.2 盈亏判定
- **买入指令**：成交均价 < 市场均价 → 盈利 ✅
- **买入指令**：成交均价 > 市场均价 → 亏损 ❌
- **卖出指令**：成交均价 > 市场均价 → 盈利 ✅
- **卖出指令**：成交均价 < 市场均价 → 亏损 ❌

### 1.3 关键约束
- 必须执行完基金经理的指令（不能漏单）
- 日结算，当日必须完成
- 考核的是相对市场均价的超额收益

---

## 二、市场均价定义

### 2.1 常见计算方式
市场均价通常指从**收到指令时刻**到**收盘**的成交量加权平均价(VWAP)

```
市场VWAP = Σ(成交价 × 成交量) / Σ(成交量)
```

### 2.2 时间窗口
- **指令接收时刻 (T)**：基金经理下发指令
- **收盘时刻**：当日交易结束
- **你的交易窗口**：T 到 收盘

## 三、最优交易策略

### 3.1 核心原则
**"逆向操作，顺势而为"**

### 3.2 买入指令策略

#### 策略A：下跌买入法（推荐）
```
条件：股价从T时刻开始下跌
操作：分批买入，越跌越买
优势：成交均价低于市场VWAP

具体做法：
1. 将指令拆分为5-10批
2. 每下跌1-2%买入一批
3. 尾盘必须完成全部指令
```

#### 策略B：尾盘抢筹法
```
条件：股价全天上涨，无回调
操作：14:30后集中买入
优势：避免追高，成交均价接近收盘价
风险：可能买不到足够数量
```

#### 策略C：震荡摊平法
```
条件：股价震荡
操作：均匀分布在全天买入
优势：成交均价接近市场VWAP，小幅盈利
```

### 3.3 卖出指令策略

#### 策略A：上涨卖出法（推荐）
```
条件：股价从T时刻开始上涨
操作：分批卖出，越涨越卖
优势：成交均价高于市场VWAP

具体做法：
1. 将指令拆分为5-10批
2. 每上涨1-2%卖出一批
3. 尾盘必须完成全部指令
```

#### 策略B：开盘冲高法
```
条件：开盘冲高
操作：开盘30分钟内集中卖出
优势：抓住早盘流动性最好的时机
```

## 四、风险控制

### 4.1 必须避免的情况

| 情况 | 后果 | 应对 |
|------|------|------|
| 买入指令追高 | 成交均价远高于市场 | 宁可尾盘买，不要追高买 |
| 卖出指令杀跌 | 成交均价远低于市场 | 宁可尾盘卖，不要杀跌卖 |
| 指令未完成 | 考核直接不及格 | 尾盘必须完成，价格次要 |

### 4.2 时间分配
```
早盘 (9:30-10:30)：观察，少量试探
午盘 (10:30-13:00)：根据趋势执行大部分
尾盘 (14:30-15:00)：必须完成剩余指令
```

## 五、实战操作手册

### 5.1 买入指令操作流程

**Step 1：接收指令**
- 记录：股票代码、买入数量、接收时间T

**Step 2：判断趋势**
- 看T时刻后的走势：上涨/下跌/震荡

**Step 3：选择策略**
```
if 下跌：
    采用 下跌买入法
    分批买入，越跌越买
    
elif 震荡：
    采用 震荡摊平法
    均匀分布在午盘买入
    
elif 上涨：
    等待回调
    若14:30仍无回调：
        采用 尾盘抢筹法
        14:30后集中买入
```

**Step 4：执行交易**
- 拆单：每批不超过指令量的10-20%
- 间隔：每批间隔5-15分钟
- 监控：实时计算当前成交均价

**Step 5：尾盘检查**
- 14:50检查完成度
- 若未完成，市价扫单完成

### 5.2 卖出指令操作流程

**Step 1-2**：同上

**Step 3：选择策略**
```
if 上涨：
    采用 上涨卖出法
    分批卖出，越涨越卖
    
elif 震荡：
    采用 震荡摊平法
    均匀分布卖出
    
elif 下跌：
    等待反弹
    若14:30仍无反弹：
        尾盘集中卖出
```

## 六、高级技巧

### 6.1 盘口语言
- **大买单托底**：可以分批买入
- **大卖单压顶**：可以分批卖出
- **对倒盘**：观望，不要参与

### 6.2 量价关系
- **价跌量缩**：底部信号，大胆买入
- **价涨量缩**：顶部信号，大胆卖出
- **放量突破**：跟随趋势

### 6.3 大盘配合
- 大盘上涨时：买入可以积极，卖出可以等待
- 大盘下跌时：买入必须谨慎，卖出必须果断

## 七、复盘与优化

### 7.1 每日复盘
```
1. 记录每只股票的：
   - 市场VWAP
   - 你的成交均价
   - 盈亏金额
   
2. 分析亏损原因：
   - 追高？杀跌？还是市场极端？
   
3. 总结经验：
   - 哪些策略有效？
   - 哪些需要改进？
```

### 7.2 持续优化
- 建立个人交易日志
- 统计不同策略的胜率
- 优化拆单比例和时间间隔
