📚 第一章 理论基础

1.1 四类资产

四元投资理论将可投资资产分为四大类别：

  1. 股票（Equity） — 经济增长的受益者
  2. 债券（Bond） — 固定收益，衰退期避风港
  3. 大宗商品（Commodity） — 通胀对冲工具
  4. 现金（Cash） — 流动性储备，滞胀期防御

1.2 经济周期四阶段

基于经济增长和通胀两个维度，经济周期分为四个阶段：

通胀↑
│
滞胀期(现金) │ 过热期(大宗商品)
│
───────────┼───────────→ 经济增长
│
衰退期(债券) │ 复苏期(股票)
│
通胀↓
阶段 经济 通胀 最优资产 配比建议
复苏期 ↑ ↓ 股票 60-70%
过热期 ↑ ↑ 大宗商品 40-50%
滞胀期 ↓ ↑ 现金 30-40%
衰退期 ↓ ↓ 债券 50-60%

1.3 单向时间箭头

核心原则：经济周期像时间一样单向流动
  • 正常顺序：复苏 → 过热 → 滞胀 → 衰退 → 复苏
  • 一旦确认进入下一阶段，绝不回头
  • 允许跳过：衰退 → 复苏（跳过滞胀）
  • 不强制补全：不纠结于缺失的阶段

⚙️ 第二章 双层调仓机制

2.1 架构设计

┌─────────────────────────────────────┐
│  第一层：四元品类调仓（月度）          │
│  股票 ↔ 债券 ↔ 大宗商品 ↔ 现金        │
│  根据周期阶段调整大类配比              │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  第二层：品类内行业调仓（月度/事件驱动）│
│  股票：科技 ↔ 消费 ↔ 周期 ↔ 金融...   │
│  大宗商品：石油 ↔ 化工 ↔ 金属 ↔ 农产品 │
│  根据微观事件调整内部结构              │
└─────────────────────────────────────┘

2.2 品类调仓

调仓频率：月度（每月1日）

触发条件：

  • 周期阶段置信度变化
  • 重大政策/地缘事件
  • 极端市场情绪

配比调整逻辑：

当前配比 = f(阶段置信度, 风险预算, 极端情绪修正)
阶段置信度 ∈ [0, 1]

2.3 行业内调仓

股票内部细分：

行业 适用阶段 触发事件
科技成长 复苏早期、流动性宽松 降息、创新突破
消费白马 复苏中期、过热期 收入预期改善
周期资源 过热期、滞胀期 通胀上行、供需紧张
金融地产 衰退后期、复苏早期 政策刺激、利率下行

📈 第五章 历史案例

2008年金融危机

2007.10

过热期 — 商品40% / 股票40% / 债券15% / 现金5%

持有有色、煤炭，通胀高企，大宗商品暴涨

2008.9

衰退期 — 债券50% / 现金40% / 股票10%

雷曼破产，清仓商品买国债，避开最惨下跌

2009.3

复苏期 — 股票60% / 债券25% / 现金15%

四万亿刺激，重仓金融地产，抓住反弹主升浪

2020年新冠疫情

2020.2

衰退期 — 现金50% / 债券40% / 股票10%

武汉封城，紧急避险，避开最恐慌时刻

2020.4

复苏期 — 股票50% / 债券30% / 现金20%

美联储无限QE，加仓科技医药，抓住大反弹

2021.2

过热期 — 股票50% / 商品30% / 债券15%

通胀抬头，加仓大宗商品，涨幅50-100%

2026年伊朗战争（当前案例）

2026.3.1

复苏期 — 股票60% / 债券20% / 商品15% / 现金5%

战争前正常配置

2026.3.10

商品期 — 商品40% / 股票30% / 债券20% / 现金10%

战争爆发，紧急调仓，股票降、商品升

2026.3.15

商品期 — 商品50% / 股票25% / 债券15% / 现金10%

油价破100美元，加仓石油至45%

👤 关于作者

周京昱

四元投资理论创始人，资深投资人。

"每次大亏，都是因为'死扛'；每次大赚，都是因为'顺势而为'。但问题是，什么是'势'？如何判断？"

2023年冬，在复盘自己的投资经历时，周京昱开始思考如何设计一个更灵活、更实用的投资框架。经过两年的打磨和实盘验证，四元投资理论从模糊的想法，变成了可执行、可验证、可复制的投资方法。

核心哲学

  1. 承认无知 — "我不知道明天市场会涨会跌，但我知道现在处于什么阶段。"
  2. 顺势而为 — "不要试图预测市场，而要识别趋势，然后跟随。"
  3. 控制风险 — "投资的本质不是赚钱，而是不亏钱。活下去，才有机会。"
  4. 保持灵活 — "没有永恒的最优资产，只有当下的最优选择。"
"理论是地图，市场是 terrain。地图可以帮助你导航，但你必须亲自走每一步。"
